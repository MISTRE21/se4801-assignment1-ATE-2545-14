## Prerequisites
Before building and running this project, make sure the following are installed on your machine:

- Java 21 or later
- Maven 3.9 or later
- Git

## How to Install

### 1. Install Java
Make sure Java 21 or later is installed.

Check if Java is installed:
java -version


### 2. Install Maven
Make sure Maven 3.9 or later is installed.

Check if Maven is installed:
mvn -version


### 3. Clone or Download the Project
If using Git:
git clone https://github.com/MISTRE21/se4801-assignment1-ATE-2545-14.git
cd <project-folder>


If you downloaded the ZIP file, extract it and open the project folder in a terminal.

## How to Build
Run the following command in the project root folder:
mvn clean install


## How to Run
Start the Spring Boot application with:
mvn spring-boot:run


The application will usually run on:
http://localhost:8080


## How to Run Tests
Run all tests with:
mvn test
